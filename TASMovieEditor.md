## Introduction ##

TAS Movie Editor is a program written to easily edit binary movie movie files (formats such as .fcm, .gmv, .vbm, etc).

Supported formats:
  * SMV
  * FCM
  * GMV
  * FMV
  * VBM
  * M64
  * MMV
  * PXM
  * PJM (binary only)
